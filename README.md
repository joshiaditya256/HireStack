# HireStack 🚀
### A Microservices-Based Professional Job Portal


![GitHub repo size](https://img.shields.io/github/repo-size/joshiaditya256/HireStack)

---

## 📌 About the Project
HireStack is a microservices-based job portal connecting job seekers 
and recruiters through a structured pipeline. It features tailored 
interfaces for candidates, recruiters, and admins.

---

## 🏗️ Architecture
HireStack is built on **5 core microservices**:
| Service | Description |
|---|---|
| 🔐 Authentication | User login, registration & security |
| 👤 Profile | Candidate & recruiter profile management |
| 💼 Job | Job posting and management |
| 📨 Feed | Personalized job recommendations |
| 🌐 API Gateway | Central entry point for all services |
